# Implementation

```javascript
class EventEmitter {
  constructor() {
    // Map structure: eventName (string) → array of listener functions
    this.listeners = new Map();
  }

  /**
   * Register an event listener
   * @param {string} eventName - The name of the event to listen for
   * @param {Function} listener - The callback function to execute when the event is emitted
   * @returns {EventEmitter} - Returns 'this' to support method chaining
   */
  on(eventName, listener) {
    // Validate inputs
    if (typeof eventName !== 'string' || eventName.trim() === '') {
      throw new TypeError('Event name must be a non-empty string');
    }
    if (typeof listener !== 'function') {
      throw new TypeError('Listener must be a function');
    }

    // Get the array of listeners for this event, or create one if it doesn't exist
    if (!this.listeners.has(eventName)) {
      this.listeners.set(eventName, []);
    }

    // Add the listener to the array
    this.listeners.get(eventName).push(listener);

    // Return 'this' for method chaining
    return this;
  }

  /**
   * Emit an event, triggering all registered listeners
   * @param {string} eventName - The name of the event to emit
   * @param {...any} args - Arguments to pass to each listener
   * @returns {boolean} - Returns true if the event had listeners, false otherwise
   */
  emit(eventName, ...args) {
    // Validate event name
    if (typeof eventName !== 'string' || eventName.trim() === '') {
      throw new TypeError('Event name must be a non-empty string');
    }

    // Check if any listeners exist for this event
    if (!this.listeners.has(eventName)) {
      return false;
    }

    // Get the listeners array and invoke each listener with the provided arguments
    const listeners = this.listeners.get(eventName);
    listeners.forEach((listener) => {
      listener(...args);
    });

    // Return true if listeners were invoked
    return listeners.length > 0;
  }

  /**
   * Remove an event listener
   * @param {string} eventName - The name of the event
   * @param {Function} listener - The specific listener function to remove
   * @returns {EventEmitter} - Returns 'this' for method chaining
   */
  off(eventName, listener) {
    // Validate inputs
    if (typeof eventName !== 'string' || eventName.trim() === '') {
      throw new TypeError('Event name must be a non-empty string');
    }
    if (typeof listener !== 'function') {
      throw new TypeError('Listener must be a function');
    }

    // If the event doesn't exist, return early
    if (!this.listeners.has(eventName)) {
      return this;
    }

    // Get the listeners array
    const listeners = this.listeners.get(eventName);

    // Find and remove the listener (handles potential duplicates by removing all occurrences)
    const index = listeners.indexOf(listener);
    if (index !== -1) {
      listeners.splice(index, 1);
    }

    // Clean up: if no listeners remain for this event, delete the entry
    if (listeners.length === 0) {
      this.listeners.delete(eventName);
    }

    // Return 'this' for method chaining
    return this;
  }
}

// Export for use in Node.js or module systems
module.exports = EventEmitter;
```

# Summary

✅ **Implementation Complete**

- **on(eventName, listener)**: Registers listeners, validates inputs, returns `this` for chaining
- **emit(eventName, ...args)**: Invokes all listeners for an event, returns boolean indicating if listeners existed
- **off(eventName, listener)**: Removes a specific listener, cleans up empty event entries, returns `this` for chaining
- **Data Structure**: Map with event names as keys, arrays of functions as values (meets architectural constraint)
- **Method Chaining**: Both `on()` and `off()` return `this`
- **Input Validation**: Type checking for event names and listeners
- **Memory Efficiency**: Removes event entries when no listeners remain

The EventEmitter is ready for use.
