## 3. Implementation

```python
import time
import functools
from typing import Callable, Type, Tuple, Union, Any

def retry(
    max_retries: int = 3,
    base_delay: float = 1.0,
    max_delay: float = 60.0,
    base: float = 2.0,
    retry_on: Union[Type[Exception], Tuple[Type[Exception], ...]] = Exception
) -> Callable:
    """
    Decorator that retries a function with exponential backoff.
    
    Args:
        max_retries: Maximum number of retry attempts (default: 3)
        base_delay: Initial delay in seconds (default: 1.0)
        max_delay: Maximum delay cap in seconds (default: 60.0)
        base: Exponential base for backoff calculation (default: 2.0)
        retry_on: Exception type(s) to catch and retry on (default: Exception)
    
    Returns:
        Decorated function that retries on specified exceptions
    
    Raises:
        The last caught exception if all retries are exhausted
    
    Example:
        @retry(max_retries=5, base_delay=0.5, max_delay=30.0)
        def unreliable_api_call():
            ...
    """
    
    # Normalize retry_on to tuple for consistent handling
    if isinstance(retry_on, type):
        retry_on = (retry_on,)
    
    def decorator(func: Callable) -> Callable:
        @functools.wraps(func)
        def wrapper(*args, **kwargs) -> Any:
            last_exception = None
            
            # Loop through retry attempts (attempt 0 is the initial call)
            for attempt in range(max_retries + 1):
                try:
                    return func(*args, **kwargs)
                
                except retry_on as e:
                    last_exception = e
                    
                    # If we've exhausted retries, re-raise the exception
                    if attempt >= max_retries:
                        raise
                    
                    # Calculate exponential backoff delay
                    delay = min(base_delay * (base ** attempt), max_delay)
                    
                    # Sleep before retrying
                    time.sleep(delay)
            
            # This line should never be reached due to the raise in the loop,
            # but included for completeness and type safety
            if last_exception is not None:
                raise last_exception
        
        return wrapper
    
    return decorator
```

## 5. Implementation Details

- **Attempt numbering:** Attempt 0 is the first try, attempt 1–N are retries. This aligns with the formula: on attempt 0, delay = `base_delay * (2^0)` = `base_delay`.
- **Exception re-raising:** When `attempt >= max_retries`, the exception is immediately re-raised without sleeping, ensuring no wasted delay after the final attempt.
- **Type safety:** Uses `functools.wraps` to preserve function metadata and provides type hints for clarity.
- **Flexible exception handling:** The `retry_on` parameter defaults to `Exception` but can be narrowed to specific types (e.g., `IOError`, `TimeoutError`).
